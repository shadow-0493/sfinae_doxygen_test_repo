void print_ip ( const T & container)

Вывод IP-адреса для контейнеров

Элементы контейнера выводятся последовательно через точку. Поддерживаются std::vector, std::list, std::set, std::map и другие.

Параметры шаблона
TТип контейнера (должен иметь методы begin() и end())
Аргументы
containerКонтейнер для вывода
